# Registration Form 📝

A simple, clean registration form built with **HTML**, **CSS**, and **JavaScript**.

## 🚀 Features

- Name, Email, Password input fields
- Gender selection with radio buttons
- Form validation
- Saves user data to `localStorage`
- Responsive and clean UI
